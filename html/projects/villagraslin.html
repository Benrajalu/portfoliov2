A propos

Site vitrine et évènementiel responsive, Villa Graslin peut être un bon exemple de ce qu'il est possible de faire pour un petit budget et des besoins simples. Malgré ces contraintes, le site reste entièrement sur-mesure afin de correspondre aux demandes précises du client.

La gestion de projet et la direction artistique est au crédit de l'agence Speederlan (SIII). L'intégration responsive est à ma charge. La rédaction web est fournie par le client.

Proposition graphique initiale

Le brief graphique original de Villa Graslin était extrêmement libre. Ma proposition reposait sur un aspect "plein écran" inspiré par la mode actuelle et les interfaces type "metro" (windows 8, Xbox...). Le logo et le code couleurs étaient alors les seuls éléments imposés. La production finale se sera inspiré de quelques éléments de cette première maquette pour coller parfaitement aux besoins du client.

Cette intégration bénéficiait d'une carte interactive Google Maps entièrement personnalisée pour correspondre au code couleur imposé. Une autre partie du dévéloppement spécifique final se retrouve dans le formulaire de contact, utilisant Ajax.
